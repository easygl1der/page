Talk Scene - Sharing LaTeX typesetting skills with students

Abstract

Introducing LaTeX fundamentals to new students, including basic functions, installation methods, usage tips, etc. This event is jointly organized by SPS Physics Association and Duxing Studio.

LaTeX is a high-quality typesetting system particularly suitable for academic papers, books, and technical document typesetting. In this talk, we systematically covered:

  • Introduction - Basic concepts and advantages of LaTeX
  • What is LaTeX for? - Application scenarios and practical value
  • Installation - Environment configuration and software installation
  • Content Writing - Document structure and basic commands
  • Formulas - Mathematical formula writing techniques
  • Personal Tips - Practical techniques and best practices
  • Links & Navigation - Cross-references and navigation
  • LaTeX Drawing - TikZ drawing functionality
  • AI for LaTeX - Modern tool assistance
  • Markdown & LaTeX - Combined use of both formats

The talk was enthusiastic with active student participation, exploring LaTeX's powerful functions in academic writing together.
